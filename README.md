# 🩺 API - Saúde Comunitária

## 📌 Sobre o projeto
Esse projeto é uma API feita em Java com o objetivo de ajudar no acesso da população a serviços básicos de saúde.

A ideia é permitir que as pessoas consigam:
- Agendar consultas
- Ver campanhas de vacinação
- Encontrar unidades de saúde
- Receber informações básicas de saúde

## 🎯 Objetivo
Muitas pessoas têm dificuldade de acessar serviços de saúde ou nem sabem quando tem campanha acontecendo.

Esse projeto tenta facilitar isso, juntando tudo em um só lugar.

## 🚀 Tecnologias usadas
- Java
- Spring Boot
- JPA / Hibernate
- MySQL (ou PostgreSQL)
- Maven
- GitHub
